# Executive Language Editor Brief

Review the draft for executive-facing language and composition.

Inspect:

- Strategic tone: direct, precise, sober, decision-oriented.
- Word choice: remove hedging that weakens true claims and remove certainty that evidence does not support.
- Sentence shape: shorter, sharper, less academic.
- Transitions: make the logic easy to follow.
- Repetition, throat-clearing, filler, and inflated language.
- Whether the headline can stand alone as the takeaway.

Prefer wording that sounds like a strong operator briefing senior leadership, not a marketing deck or an academic paper.

Return the shared reviewer output contract from `references/reviewer-briefs.md`, with multiple rewording options when the sentence is high-leverage.
