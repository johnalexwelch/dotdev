# Removal Pass Brief

Review the draft for what to cut, compress, or demote.

Inspect:

- Repeated ideas.
- Background that should move below the answer.
- Caveats that are real but over-presented.
- Detail that belongs in an appendix, footnote, or source note.
- Sentences that sound impressive but do not change the conclusion.
- Paragraphs that slow the reader before the main answer lands.

Return specific removals and compressions. Preserve necessary caveats and evidence.

Return the shared reviewer output contract from `references/reviewer-briefs.md`.
