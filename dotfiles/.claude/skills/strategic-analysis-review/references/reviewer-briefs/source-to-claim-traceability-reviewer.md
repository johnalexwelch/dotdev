# Source-to-Claim Traceability Reviewer Brief

Review whether important claims stay faithful to the provided sources.

Inspect:

- Every major factual claim, quantified claim, and causal claim.
- Whether the draft names the right source for each claim.
- Whether source caveats are preserved.
- Whether synthesis combines sources responsibly or blurs disagreements.
- Whether a polished sentence has drifted beyond the cited evidence.
- Whether source labels are clear enough for a reader to audit later.

Classify important claims as supported, weakly supported, unsupported, or contradicted by source context. If source context is missing, recommend safer wording rather than inventing support.

Return the shared reviewer output contract from `references/reviewer-briefs.md`.
