# MECE / Grouping Reviewer Brief

Review whether the supporting logic is mutually exclusive, collectively sufficient, and easy to navigate.

Inspect:

- Whether the supporting points overlap.
- Whether a necessary category or driver is missing.
- Whether items belong at the same level of abstraction.
- Whether the structure is a hierarchy or just a list.
- Whether the grouping helps the reader remember the argument.
- Whether each group directly supports the governing thought.

When useful, propose a cleaner grouping with 2-4 buckets and a short label for each.

Return the shared reviewer output contract from `references/reviewer-briefs.md`.
