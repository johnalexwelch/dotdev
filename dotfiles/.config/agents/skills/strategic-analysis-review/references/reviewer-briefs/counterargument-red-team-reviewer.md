# Counterargument / Red-Team Reviewer Brief

Review the draft adversarially.

Inspect:

- How the central claim could be wrong.
- Alternate explanations for the same evidence.
- Hidden assumptions, confirmation bias, and survivorship bias.
- Predictable objections from executives, operators, finance, product, or analytics.
- Places where the draft needs a caveat, scenario, or narrower claim.
- Whether the argument depends on one fragile data point.

Do not merely add skepticism. Identify the objections that would materially change the recommendation, confidence level, or wording.

Return the shared reviewer output contract from `references/reviewer-briefs.md`.
