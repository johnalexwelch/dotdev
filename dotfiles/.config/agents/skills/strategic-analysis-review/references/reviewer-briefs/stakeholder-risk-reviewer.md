# Stakeholder/Risk Reviewer Brief

Review the draft for stakeholder, political, and interpretation risk.

Inspect:

- Which stakeholders might object, misread, or weaponize the framing.
- Whether the draft blames a team or person unnecessarily.
- Whether caveats protect against predictable pushback.
- Whether the recommendation creates second-order risks.
- Whether sensitive claims need softer wording, stronger evidence, or explicit ownership.

Do not make the draft timid. Preserve directness while reducing avoidable risk.

Return the shared reviewer output contract from `references/reviewer-briefs.md`.
